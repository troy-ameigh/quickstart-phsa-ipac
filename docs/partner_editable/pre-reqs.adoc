== Data schema requirements

Bulk patient data ingested by the deployment must be in a Microsoft Excel spreadsheet (.xlsx). The spreadsheet must contain three sheets, each containing the columns and data shown in the following tables. Items in bold are required. All others are optional but recommended.

=== Sheet 1

[cols=3*]
|=== 
|*Column name*|*Data type*|*Description*
|*encntr_num*|Int64|Encounter number.
|*mrn*|Int64|Medical record number.
|encntr_id|Int64|Encounter ID.
|collection_date_id|Int64|Blood collection date.
|loc_bed_desc_src_at_collection|Int64|Patient bed number at blood collection.
|transfer_in_to_collect|Int64|Number of days from nursing unit admission date to blood collection date.
|transfer_out_to_collect|Int64|Number of days from blood collection date to patient discharge date.
|ce_dynamic_label_id|Int64|Internal ID.
|line_tube_drain_insertion_seq|Int64|Central line insertion sequence.
|line_insert_to_collection|Int64|Number of days from central line insertion date to blood collection date.
|birth_date_id|Int64|Patient date of birth.
|line_remove_to_collect|Int64|Number of days from central line removal to blood collection date in days
|bc_phn|Int64|Provincial Health Care health number
|nursing_unit_short_desc|String|Nursing unit code.
|facility_name_src|String|Hospital or clinic name at blood collection.
|encntr_type_desc_src_at_collection|String|Encounter type at blood collection.
|clinical_event_code_desc_src|String|Collection type. Enter `Blood culture.`
|loc_room_desc_src_at_collection|String|Patient room information at the blood collection.
|disch_disp_desc_src|String|Discharge disposition.
|lab_result|String|Lab result of the blood collection, either `Positive` or `Negative`.
|med_service_desc_src_at_collection|String|Medical service at collection.
|nursing_unit_desc_at_collection|String|Nursing unit name.
|nursing_unit_short_desc_at_collection|String|Nursing unit code description.
|organism_1_desc_src|String|First organism found in blood collection.
|organism_2_desc_src|String|Second organism found in blood collection.
|organism_3_desc_src|String|Third organism found in blood collection.
|result_interpretation_desc_src|String|Lab result interpretation.
|specimen_type_desc_src|String|Lab specimen type description.
|doc_set_name_result|String|Label for central line.
|first_catheter_type_result|String|Central line catheter type.
|first_site_result|String|Body site of catheter insertion.
|name_last|String|Patient last name.
|name_first|String|Patient first name.
|gender_desc_src|String|Patient gender.
|home_addr_patient_postal_code_ forward_sortation_area|String|First three digits of patient postal code.
|*admit_dt_tm*|DateTime|Hospital admission date, in format YYYY/MM/DD.
|*disch_dt_tm*|DateTime|Hospital discharge date, in format YYYY/MM/DD
|*collection_dt_tm*|DateTime|Blood collection date, in format YYYY/MM/DD.
|*beg_effective_dt_tm*|DateTime|Nursing unit admission date, in format YYYY/MM/DD.
|*end_effective_dt_tm*|DateTime|Nursing unit discharge date, in format YYYY/MM/DD.
|*first_activity_start_dt_tm*|DateTime|Central line insertion date, in format YYYY/MM/DD.
|*last_activity_end_dt_tm*|DateTime|Central line removal date, in format YYYY/MM/DD.
|===
